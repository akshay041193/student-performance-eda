Exploratory Data Analysis on Students Performance Dataset

Project Overview

This project performs Exploratory Data Analysis (EDA) on the Students Performance in Exams dataset.
The goal is to analyze how different demographic and educational factors affect student scores in Math, Reading, and Writing.

This project demonstrates:
	•	Data cleaning and exploration
	•	Statistical analysis
	•	Data visualization using Python libraries
	•	Insight generation from educational data

Dataset Information
	•	Dataset Name: Students Performance in Exams
	•	Source: Kaggle
	•	Link: https://www.kaggle.com/datasets/spscientist/students-performance-in-exams
	•	File Type: CSV
	•	Rows: 1000
	•	Columns: 8

This dataset contains student exam scores along with demographic and background attributes such as gender, parental education, lunch type, and test preparation.

Objectives of the Analysis
	•	Understand score distributions across subjects
	•	Compare performance based on gender, lunch type, and test preparation
	•	Analyze the impact of parental education on student performance
	•	Identify high-performing student groups
